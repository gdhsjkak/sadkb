# سوق VIP Scripts

موقع إلكتروني لعرض وبيع سكربتات وأدوات وسيارات وهواتف وخوادم FiveM.

## المميزات

- واجهة مستخدم عصرية وجذابة
- تصنيف المنتجات حسب الفئات
- نظام بحث متقدم
- نظام دفع آمن
- لوحة تحكم للمستخدمين
- نظام تقييم ومراجعات

## التقنيات المستخدمة

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Axios

## متطلبات التشغيل

- Node.js 18 أو أحدث
- npm أو yarn

## طريقة التشغيل

1. قم بتثبيت التبعيات:
```bash
npm install
# أو
yarn install
```

2. قم بتشغيل خادم التطوير:
```bash
npm run dev
# أو
yarn dev
```

3. افتح المتصفح على العنوان:
```
http://localhost:3000
```
